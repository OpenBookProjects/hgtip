要用Git Diffs

posted by Steve Losh on October 22, 2009

类似 hg diff 命令进行的Mercurial 差异输出格式,默认使用的是 UNINX 中 patch 的格式. 俺猜你习惯了这种格式,不过有更好的.

Git在其 diff 命令中创造了一种更加易读的差异格式, Mercurial 也可以使用之.

Mercurial维基 中有说怎么用, 不过那篇文章忒复杂了,叫俺来说,就酱 编辑你的 ~/.hgrc  包含:

[diff]
git = True

齐活! 现在 Mercurial 的差异输出就用了更好的格式. 比如,你用 hg rename 重命名了文件, 差异汇报就是 “file X renamed to Y”,而不会是原先分开的 remove 和 add 的说明.
