Keeping Informed with Progress

posted by David Harrigan on March 24, 2010

Sometimes when using Mercurial it seems to sit at the command prompt for quite some time as it churns over some files. Thankfully, since Mercurial 1.5 you can now take advantage of a new extension! You’ll want to edit your ‘~/.hgrc’ file to contain something like this:
